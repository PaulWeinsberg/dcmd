# Requirements

## Operating System

The following operating systems are supported:

- Windows 10 / 11 (Docker Desktop + WSL2 home or pro)
- MacOS 10.15 or later (Docker Desktop)
- Linux major distributions (docker-ce)

## Hardware

The CLI is available for the following architectures :

- x86_64 (amd64)
- aarch64 (arm64)
